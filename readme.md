Flat 1.0 for SemanticScuttle 0.98.5.
------------------------------------

by Roman Lehnhof
http://www.lehnhof.net

###Installation

1. Unzip theme and template
2. Put both "flat" folders in place
3. Open config.php, scroll to bottom
4. $theme = 'flat';
5. Have fun!
